Yggdrasil Network is a decentralized, end-to-end encrypted mesh network for

routing IPv6 traffic within the 0200::/7 range. Users can freely generate

these addresses, with traffic transparently routed through network participants.

Most IPv6-compatible applications work seamlessly with Yggdrasil.

For hassle-free security, use Yggstack to access the network safely without

exposing ports or needing administrative privileges.

A convenient install script is available here.

To join the public network, select a peer from the list of public peers or

use our uplink server below. For optimal performance, choose a peer

that is geographically close to your machine.
